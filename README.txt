B+ Programming Language (v1.0)

Welcome to B+ — a custom programming language created by Srinjon 🚀

========================
HOW TO RUN
==========

1. Open Command Prompt / PowerShell
2. Navigate to this folder
3. Run:

   BPlus.exe yourfile.bp

========================
EXAMPLE
=======

[SYSTEM] Import -- bp-dir /in

write(String("Hello World"))

========================
FEATURES
========

✔ Variables (Int, String, Dcm, Char)
✔ Math operations (ad, su, mu, di)
✔ Conditions (if, else if, else)
✔ Loops (repeat, while)
✔ Arrays (/Settings system)
✔ Functions (basic)
✔ File creation
✔ System boot engine

========================
SYNTAX EXAMPLES
===============

Variables:
x Int = 10
name String = "B+"

Math:
x Int = 5 ad 3

Repeat Loop:
repeat (3) {
write(String("Hi"))
}

Array:
/Settings() {
array[1, 2, 3]
return array as x
}
